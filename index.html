CHI 2026 Supplement


Plain-language summary. This supplement explains how yearly topological analysis can detect boundary reorganization in HCI concept space before a stable topic label becomes dominant.

Technical summary. For each year t, we build a concept embedding space from papers available up to year t, compute a Vietoris–Rips filtration, extract persistent H1 loops, reconstruct representative cycles at loop birth, detect cross-community edges on those cycles, and rank bridge candidates for temporal validation.

One-screen overview

  • Data: ~30,000 papers across CHI, UIST, CSCW, UbiComp.
  • Temporal span: 1999–2025 (CSCW available through 2024).
  • Yearly construction: each year is recomputed with a historical cutoff at year t.
  • Primary output: ranked bridge candidates from representative cycle cross-community edges.
  • Evaluation horizon: future-year relevance with nDCG@50 and lead-time interpretation.

How to read this supplement

Read Chapters 1→9 in order. The structure is evidence-first: (1) audit and research question, (2) data and mathematics, (3) bridge extraction and ranking, (4) case walkthroughs, (5) claim boundaries and reference integrity.

Reviewer shortcut: if you need implementation logic quickly, start at Chapters 4–6, then read Chapters 7–8 for interpretation constraints.
