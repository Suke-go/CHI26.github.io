Reading note. A loop means concepts surround a gap. Persistence means that pattern stays visible for longer thresholds.

Use this chapter as a plain-language guide. Birth means a boundary becomes visible. Persistence means the same unresolved shape stays visible while the threshold grows. Death (closure) means added links fill the gap so the loop is no longer open [1][2].

A practical way to read yearly outputs: first inspect one cycle in one year, then check whether it is short-lived or persistent, then compare yearly summaries. This is a "compare-yearly-states" reading, not a "single object moving through all years" reading.

Representative cycles are there to make topology inspectable. They provide a way to read the figure with basic confidence, while keeping stronger historical claims for later chapters [3].
