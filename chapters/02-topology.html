C. Core idea

Plain-language explanation. We look for loop-shaped boundary structure in each year’s concept neighborhood graph. When a reconstructed loop crosses semantic communities, its cross-community edges are treated as interpretable bridge candidates: relations worth attention because they connect nearby but still separated regions.

Technical explanation. For each year t, persistent H₁ features are computed on a Vietoris–Rips filtration over concept distances. A representative cycle is reconstructed at loop birth. Community labels on a separation graph are then used to retain only cycle edges (u,v) with c_t(u) ≠ c_t(v), yielding bridge-candidate edges for ranking and temporal validation.

Persistent homology (quick definition). Persistent homology tracks which topological features appear and disappear as the distance threshold grows. For H₁, the feature of interest is a loop-like hole. Persistence length indicates how long that hole remains unresolved across thresholds.
Representative cycle. A representative cycle is a readable witness for one loop feature. In this project it is reconstructed from a birth edge plus a pre-birth shortest path. It supports interpretation, but it is not the only valid witness for the same homology class.
