Representative cycle reconstruction rule

Plain-language layer: We reconstruct one readable cycle by combining the birth edge with the shortest path that existed just before that edge appeared.

Technical layer: For birth edge e_b = (u,v) at b_k, define representative cycle C_k = e_b ∪ P_{b_k^-}(u,v), where P_{b_k^-} is a shortest path in the graph immediately before birth.
