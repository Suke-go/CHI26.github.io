5-1. From distances to filtration

The angular distance matrix defines concept proximity. A Vietoris–Rips filtration is built by adding edges in order of increasing distance, constructing simplicial complexes at each threshold ε.

Equations

dij = 1 − cos(vi, vj) (cosine distance)

Rε(Vt) = {σ ⊆ Vt | dij ≤ ε for all i,j ∈ σ}

Computational optimization: Flag complex of k-NN graph (k=15) approximates the full Rips complex.

5-2. Persistent H₁ and structural gaps

Persistent homology computes H₁ features: loop-like holes that are born and die as ε grows. Each feature has an interval [bk, dk) and persistence πk = dk − bk. Features with π ≥ 0.05 are retained as robust structural signals.

5-3. Birth edge and representative cycle

The birth edge eb = (u, v) is the edge whose addition creates the loop. The representative cycle is:

Ck = eb ∪ P(u, v)

where P(u, v) is the shortest path between u and v in the graph just before birth. This cycle provides an interpretable chain of intermediate concepts.

5-4. Bridge scoring

Bridge candidates are cycle edges that cross community boundaries:

Ebridget = {(i,j) ∈ Ck | ct(i) ≠ ct(j)}

Each edge is scored by the maximum persistence of any H₁ cycle it participates in. Top-50 edges by persistence are output for evaluation.

5-5. Yearly topology summaries

At the macro level, yearly summaries include: total persistence (sum of all H₁ feature persistence), number of persistent features, and triadic closure rate. These characterize the field's structural state: whether it is boundary-heavy (many unresolved loops) or closure-heavy (loops being filled).
