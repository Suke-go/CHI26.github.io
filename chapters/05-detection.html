5-1. From distances to filtration

The angular distance matrix defines concept proximity. A Vietoris–Rips filtration is built by adding edges in order of increasing distance, constructing simplicial complexes at each threshold ε.

Equations

dij = 1 − cos(vi, vj) (cosine distance)

Rε(Vt) = {σ ⊆ Vt | dij ≤ ε for all i,j ∈ σ}

Computational optimization: Flag complex of k-NN graph (k=15) approximates the full Rips complex.

5-2. Persistent H₁ and structural gaps

Persistent homology computes H₁ features: loop-like holes that are born and die as ε grows. Each feature has an interval [bk, dk) and persistence πk = dk − bk. Features with π ≥ 0.05 are retained as robust structural signals.

5-3. Birth edge and representative cycle

The birth edge eb = (u, v) is the edge whose addition creates the loop. The representative cycle is:

Ck = eb ∪ P(u, v)

where P(u, v) is the shortest path between u and v in the graph just before birth. This cycle provides an interpretable chain of intermediate concepts.

5-3b. Words, not concepts: what intermediaries actually represent

A common misreading is to interpret intermediaries as conceptual mediators—ideas that theoretically bridge two fields. They are not. Intermediaries are stepping stones in Word2Vec embedding space: each step connects to the next by vector proximity in the trained model, not by any semantic or theoretical relationship.

Key distinction. The representative cycle is a path through language space (how words are used in paper abstracts), not through concept space (how ideas relate in a theoretical framework). Sometimes these align; often they do not.

When language paths are interpretable

speaker → smart (2020, integrated, 8 concepts):

speaker → conversation → partner → friend → family → household → home → smart

This path traces a natural semantic gradient from audio/speech through social relationships to domestic computing. Each step is linguistically close to the next, and the overall trajectory is interpretable as “voice interfaces entering the home.”

effective → feedback (2018, integrated, 6 concepts):

effective → efficiency → latency → force → haptic → feedback

Short path with a coherent trajectory: performance metrics → physical interaction → haptic feedback. The linguistic gradient matches a plausible conceptual connection.

When language paths are not interpretable

security → privacy (2017, integrated, 14 concepts):

security → authentication → scheme → accurate → vehicle → driver → interruption → stress → mental → health → woman → attitude → concern → privacy

Security and privacy did integrate (max co-occurrence = 9), but the path passes through automotive, mental health, and gender studies. These are not conceptual mediators—they are simply nearby words in the embedding space that happen to form the shortest distance chain.

novice → programming (2019, decayed, 22 concepts):

novice → workflow → manual → automatic → algorithm → prediction → metric → bias → gender → facebook → twitter → political → collective → stakeholder → participatory → workshop → studio → toolkit → hardware → software → developer → programming

The longest cycle in the dataset (22 concepts). The path wanders through AI/ML, social media, civic tech, and maker culture before reaching programming. This bridge decayed—no coherent integration trajectory existed despite the structural detection.

Summary: intermediaries as signal quality indicator

Path interpretability is not guaranteed by the method. It is an additional quality check that readers should apply when evaluating bridge candidates. A short, coherent path strengthens the bridge hypothesis; a long, wandering path weakens it.

5-4. Bridge scoring

Bridge candidates are cycle edges that cross community boundaries:

Ebridget = {(i,j) ∈ Ck | ct(i) ≠ ct(j)}

Each edge is scored by the maximum persistence of any H₁ cycle it participates in. Top-50 edges by persistence are output for evaluation.

5-5. Yearly topology summaries

At the macro level, yearly summaries include: total persistence (sum of all H₁ feature persistence), number of persistent features, and triadic closure rate. These characterize the field's structural state: whether it is boundary-heavy (many unresolved loops) or closure-heavy (loops being filled).
