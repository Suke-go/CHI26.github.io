E. Mathematical pipeline

\( d_{ij} = 1 - \dfrac{v_i^{\top}v_j}{\lVert v_i \rVert_2 \lVert v_j \rVert_2} \)

This defines concept distance from vector similarity: nearer concepts have smaller dᵢⱼ, and these distances drive all later topology.

\( \mathcal{R}_{\varepsilon}(V_t) = \{ \sigma \subseteq V_t \mid d_{ij} \le \varepsilon,\ \forall i,j \in \sigma \} \)

This is the Vietoris–Rips complex at threshold ε: as ε increases, more edges and simplices appear.

\( H_1(\mathcal{R}_{\varepsilon}(V_t)) \)

H₁ captures one-dimensional loop structure in the complex at each threshold.

\( I_k = [b_k, d_k),\ \pi_k = d_k - b_k \)

For loop feature k, bk is birth, dk is death, and πk is persistence length. Longer persistence indicates a loop that remains unresolved over a wider threshold range.

\( C_k = e_b \cup P_{b_k^-}(u,v) \)

This representative cycle combines the birth edge eb=(u,v) with a shortest path just before birth. We use the same representative-cycle definition introduced in Chapter 2.

Notation used in Chapters 4–6

  • V_t: concept set built from data available up to year t.
  • d_{ij}: distance between concept i and j.
  • \(\mathcal{R}_{\varepsilon}(V_t)\): Vietoris–Rips complex at threshold ε.
  • I_k=[b_k,d_k): birth/death interval of loop feature k.
  • C_k: representative cycle used for interpretation and extraction.
