7-1. What “cross-area link” means here

In this project, a “cross-area link” means a pair of concepts that belong to different Louvain communities in the year-specific semantic graph. It does not mean:

  • Institutional interdisciplinarity (departments collaborating)
  • Author-level boundary crossing
  • Funding-stream overlap

The communities are semantic, not social or institutional.

7-2. Reading cycle birth, persistence, and death

EventMeaningWhat it suggests
Birth A loop closes in concept space A boundary structure has formed: communities approach without merging
Persistence The loop survives across thresholds The boundary is structurally robust, not an artifact of a specific distance cutoff
Death A simplex fills the loop interior Possible resolution: either integration (concepts merge) or structural collapse

7-3. Reading representative cycles

Reminder from Ch. 5. Intermediaries are stepping stones in Word2Vec embedding space, not conceptual mediators. The cycle path shows how words are linguistically related in paper abstracts, not how ideas are theoretically related. Sometimes these align—and when they do, the bridge is more credible.

Anatomy of a representative cycle

ComponentDefinitionRole
EndpointsThe two words of the birth edge (u, v)The bridge pair itself—the structurally most distant pair in the cycle
IntermediariesAll path nodes between u and v (excluding endpoints)The embedding-space stepping stones. Short, coherent chains suggest a real field adjacency; long, wandering chains suggest a distance artifact
Representative cycleBirth edge + shortest path before birthOne computable witness for the H₁ class. It is not the only valid decomposition

Integrated bridges: what the cycles look like

VR – Therapy (2013, integrated in 3 years, 5 concepts):

vr → immersion → exergames → therapies → therapy

Short path (3 intermediaries). The gradient is interpretable: VR technology → immersive experience → exercise games → therapeutic applications. The boundary concept “exergames” sits exactly where design-technical and clinical communities meet.

speaker → smart (2020, integrated in 1 year, 8 concepts):

speaker → conversation → partner → friend → family → household → home → smart

A natural gradient from audio/speech through social relationships to domestic IoT. The path traces “voice interfaces entering the home”—a trajectory that was indeed realized in smart speaker research.

effective → feedback (2018, integrated in 1 year, 6 concepts):

effective → efficiency → latency → force → haptic → feedback

The shortest integrated cycle (4 intermediaries). Performance metrics → physical interaction → haptic feedback. Linguistic proximity mirrors conceptual proximity.

simulation → training (2017, integrated in 2 years, 9 concepts):

simulation → immersive → virtual → presence → sense → emotional → mental → exercise → training

VR simulation connects to training through embodied experience and exercise—reflecting the emerging VR training applications.

Decayed bridges: what went wrong

novice → programming (2019, decayed, 22 concepts):

novice → workflow → manual → automatic → algorithm → prediction → metric → bias → gender → facebook → twitter → political → collective → stakeholder → participatory → workshop → studio → toolkit → hardware → software → developer → programming

The longest cycle in the dataset (20 intermediaries). The path wanders through AI/ML, social media, civic tech, and maker culture. No coherent integration trajectory exists—the bridge was a distance artifact in the embedding space.

tutorial → live (2017, decayed, 15 concepts):

tutorial → editing → sequence → manual → layout → intuitive → hybrid → construction → studio → creativity → meaning → meaningful → narrative → story → live

The path meanders through UI design, creativity, and narrative—never forming a coherent connection between tutorials and live experiences.

friend → conversation (2017, decayed, 13 concepts):

friend → facebook → strength → password → scheme → continuous → capable → projection → immersive → avatar → presence → partner → conversation

Despite “friend” and “conversation” seeming intuitively related, the shortest embedding path traverses social media, security, and VR. The linguistic relationship exists, but the structural relationship in the HCI corpus passes through unrelated fields.

Practical reading heuristic

SignalStronger bridge hypothesisWeaker bridge hypothesis
Path lengthShort (4–8 intermediaries)Long (10+ intermediaries)
Semantic coherenceSteps follow a recognizable research trajectorySteps wander through unrelated fields
Intermediary qualityDomain-specific nouns (immersion, haptic, household)Function words or generic terms (capable, meaningful, scheme)
Endpoint specificityBoth endpoints are domain conceptsOne or both endpoints are generic (effective, along)
Bottom line. A representative cycle is a hypothesis generator, not a proof. Short, coherent cycles with domain-specific intermediaries point to real field adjacencies. Long, wandering cycles with generic words are distance artifacts. The method detects both; the reader must distinguish them.
